# Política de Privacidade

**Última atualização:** Junho de 2026

O aplicativo **Álbum da Copa** respeita a privacidade dos usuários.

## 1. Dados Coletados

O aplicativo pode armazenar as seguintes informações:

* Nome
* E-mail
* Senha cadastrada
* Status das figurinhas coletadas

## 2. Como os Dados São Utilizados

Os dados são utilizados exclusivamente para:

* Autenticação do usuário
* Identificação da conta
* Organização das figurinhas do álbum

## 3. Armazenamento das Informações

As informações do aplicativo são mantidas localmente em memória, sem integração com banco de dados externo.

## 4. Compartilhamento de Dados

O aplicativo **não compartilha informações pessoais com terceiros**.

## 5. Segurança

Embora medidas básicas sejam utilizadas no desenvolvimento do aplicativo, nenhuma plataforma é totalmente livre de riscos.

## 6. Alterações na Política

Esta política poderá ser atualizada futuramente para refletir melhorias no sistema.

Ao utilizar o aplicativo, você concorda com esta política de privacidade.
